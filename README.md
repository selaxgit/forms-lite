# Forms lite

Замена U-Forms без привязки к конкретным приложениям.

[Демо](https://selaxgit.github.io/forms-lite)

Фреймворк: Angular

Хранение данных: IndexedDB

### Библиотеки

- Angular material
- @ngrx/component-store

### Установка и запуск

```shell
$ npm install
$ npm run start
```

### Сборка Electron

```shell
$ ./build-electron.sh
```
